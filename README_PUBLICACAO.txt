COMO PUBLICAR O SITE COGNUS

OPCAO 1: Netlify, mais simples
1. Acesse netlify.com e crie uma conta.
2. Clique em Add new site e depois Deploy manually.
3. Arraste a pasta cognus_site ou o arquivo cognus_site.zip.
4. Aguarde o deploy e copie o link gerado.
5. Para usar dominio proprio, entre em Domain settings e adicione o dominio.
6. No provedor do dominio, ajuste os DNS conforme o Netlify informar.

OPCAO 2: GitHub Pages
1. Crie uma conta no github.com.
2. Crie um repositorio chamado cognus-site.
3. Envie os arquivos index.html e LOgo Cognus.jpeg.
4. Entre em Settings, Pages.
5. Em Build and deployment, escolha Deploy from a branch.
6. Selecione main e root.
7. Aguarde o link ser gerado.

OPCAO 3: Publicar em hospedagem tradicional
1. Contrate uma hospedagem com acesso ao gerenciador de arquivos ou FTP.
2. Envie index.html e LOgo Cognus.jpeg para a pasta public_html ou www.
3. Acesse o dominio para testar.

OBSERVACAO SOBRE O FORMULARIO
O formulario atual usa mailto para abrir o aplicativo de email do visitante.
Para receber mensagens direto no email sem depender do app do visitante, integre com Formspree, Netlify Forms ou outro serviço de formulario.
